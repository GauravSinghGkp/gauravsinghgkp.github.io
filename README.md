# GauravDev - Developer Website

Personal developer portfolio website for hosting app-ads.txt and showcasing my apps.

## Purpose

- **AdMob Verification**: Hosts `app-ads.txt` for Google AdMob app verification
- **Developer Portfolio**: Showcases my published Android/React Native apps
